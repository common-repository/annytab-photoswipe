A plugin that implements PhotoSwipe to display an image gallery in a lightbox.

== Description ==

This plugin implements [PhotoSwipe](https://photoswipe.com/) to display an image gallery in a lightbox. To include images in the image gallery, you need to link them to "Media File". The lightbox opens when you click on an image.

== Installation ==

Upload the plugin and activate it. You do not need to set any configuration options.

== Screenshots ==

1. Shows 1 image of 4 in an image gallery, the lightbox is shown in fullscreen mode.
